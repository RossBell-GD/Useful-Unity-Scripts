# Useful-Unity-Scripts
A collection of reusable C# scripts that I use across various Unity projects. The repository covers a range of systems, including core gameplay mechanics, UI functionality, performance settings, target frame rates, and other commonly used features.

While I cannot guarantee that every script is fully optimised for every project, each one includes comments and a brief explanation to make it easier to understand, modify, and integrate into your own games.

This is an open repository, meaning you are free to clone, use, and edit any of the content in both commercial and non-commercial projects.

Credit is not required. However, I would love to see what you create using these scripts. You can tag my itch.io account at https://rdbi.itch.io/ or contact me by email!
